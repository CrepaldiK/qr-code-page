## Meu processo

### Tecnologias utilizadas

- HTML5 semântico
- CSS3 customizado
- Flexbox para layout responsivo

### O que eu aprendi

Neste projeto, pratiquei a criação de um componente centralizado e responsivo utilizando Flexbox. Um dos principais aprendizados foi trabalhar com a centralização vertical e horizontal de elementos:

```css
body {
    display: flex;
    justify-content: center;
    align-items: center;
    height: 100vh;
}
```

Também apliquei conceitos de design como:
- Uso de `border-radius` para criar cantos arredondados suaves
- Aplicação de `box-shadow` para dar profundidade ao card
- Hierarquia visual através de diferentes pesos de fonte e cores

### Desenvolvimento contínuo

Em projetos futuros, pretendo:
- Implementar media queries para maior responsividade
- Explorar CSS Grid para layouts mais complexos
- Adicionar animações e transições
- Trabalhar com variáveis CSS para melhor manutenibilidade
